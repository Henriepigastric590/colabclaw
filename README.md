# ColabCraw

Google Colab에서 OpenClaw를 실험적으로 사용해보기 위한 예제 저장소입니다.

## 목표

- Colab 환경에서 OpenClaw 설치 가능 여부 확인
- 짧은 데모/튜토리얼 페이지 제공
- 추후 예제 노트북과 문제 해결 가이드 추가

## 포함 내용

- `index.html` : 간단한 소개 랜딩 페이지
- `docs/email-issue-workflow.md` : Gmail → GitHub Issue → 답장 초안 워크플로 문서
- `.github/ISSUE_TEMPLATE/email_from_gmail.md` : 메일 이슈 템플릿
- `.github/labels-plan.md` : 라벨 설계안
- `.github/comment-templates.md` : 초안/발송 로그 코멘트 템플릿

## 배경

OpenClaw는 보통 로컬 환경이나 서버 환경에서 쓰는 편이 더 자연스럽지만,
Colab에서는 빠른 실험, 튜토리얼 제작, 개념 검증 용도로 활용할 수 있습니다.

다만 Colab은 세션 기반이라 장기 실행에는 적합하지 않을 수 있습니다.
